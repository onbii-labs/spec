# 0019: Specification And Documentation Use CC BY 4.0

Status: Accepted

Date: 2026-07-21

## Context

Onbii is intended as an open specification that others can implement, reference, extend, and discuss. The specification and documentation do not need the same license as future code.

The project also needs to distinguish the open specification from the Onbii name and brand.

## Decision

The specification and documentation should use Creative Commons Attribution 4.0 International.

Future code repositories can choose an appropriate software license separately. The Onbii name and logo are not granted under the documentation license.

## Consequences

- The specification can be freely shared and adapted with attribution.
- Future implementations can use Apache 2.0, MPL 2.0, or another suitable code license.
- Repository notices should keep the licensing and trademark distinction clear.
- The specification remains easy for others to adopt.
