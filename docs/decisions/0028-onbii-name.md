# 0028: Project Name Is Onbii

Status: Accepted

Date: 2026-07-21

## Context

The naming discussion considered several invented names. The project needed a name that was short, pronounceable, distinctive, and not locked to one implementation category such as AI, memory, notes, recorder, graph, or operating system.

The name also needed to work as an umbrella for a specification, future tools, adapters, website, and possible ecosystem components.

## Decision

The project name is Onbii.

The GitHub organisation uses Onbii Labs as the project umbrella.

## Consequences

- Onbii can remain broad enough for an open specification and ecosystem.
- Repository names should describe concrete responsibilities, such as `spec`, `adapters`, or `website`, rather than overloading the project name.
- The Onbii name and brand should remain distinct from the open specification license.
